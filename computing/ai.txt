□ Azure Cognitive Services
機械学習の専門知識がない利用者でも手軽に利用できる、学習済のモデルをAPIde利用できるサービス
視覚、言語、音声、意思決定、検索などのカテゴリーのAPIがある

□ Azure Bot Serviceとの組み合わせ
チャットにおいてシステムが自動応答するボットの作成、開発、公開などを行うことができるマネージドサービス
Cognitive Servicesと組み合わせることで高度なボットが開発できる

□ Azure OpenAI Services
OpenAI社が地峡する　ChatGPTなどのサービスをAzure上から利用できるサービス
利用者のデータはAIモデルのトレーニングには利用されず、またAzure内のリソースからはプライベートなネットワークでセキュアにアクセスできる

□ Azure Machine Learning
機械学習モデルの作成、トレーニング、デプロイ、管理、監視を提供するマネージドサービス
PyTorchやTensorFlowなどのオープンソースを利用することが可能

□ Azure Machine Learningデザイナー
ドラッグアンドドロップでデータセットとモジュールを直感的に接続することができる学習モデル作成できる機能

□ Notebook
Azure Machine LearningからJupyter Notebookを利用できる

□ AutoML 自動機械学習
機械学習モデルの開発において多くの時間を要する反復的なタスクを自動化する機能。

□ Power Apps
スマートフォンやPC向けの高度なアプリをローコードで開発できるプラットフォーム
270をこえるサードパーティ製のプラットフォームとの接続に対応
