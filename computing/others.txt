□ Azure Container Incetances ACI
サーバーの管理なしでコンテナ環境を実行できるサービス

□ Azure Kubernetes Service AKS
Azure上でKubernetes環境を実行できるマネージドサービス

□ Azure Virtual Desktop AVD
Azure上で仮想デスクトップ環境利用できるサービス
VDI環境を構築るすためにはインフラ環境の準備が必要だが、AVDはAzureがインフラ環境甘露を行ってくれる

□ Azure Batch
大規模なバッチジョブを管理、実行してくるマネージドサービス。
コンピューティングリソースは何千台もの仮想マシンに拡張できる

□ Azure Logic Apps
ジョブやワークフローを作成できるサーバーレスの実行環境
Logic Appのデザイナー機能を使えばコードを一行も書くことなく、グラフィカルにワークフローの実行環境を作成できる

□ Azure Cognitive Services
開発者が直接的な人工知能 (AI) またはデータ サイエンスのスキルや知識がなくてもコグニティブかつインテリジェントなアプリケーションを構築できる、クラウドベースの AI サービス

□ Azure Active Diretory
セキュリティトークンの発行可能

□ Azure Service Health
障害が発生した場合の通知を受け取るルールの作成可能
環境内のサービずの正常性の確認可能

□ 管理グループ
管理グループは複数のサブスクリプションをグループ化して、効率的な管理を実施するためのグループです。
管理グループを複数のサブスクリプションの上位に設定することで、ユーザーに対して複数のサブスクリプションへのアクセスを提供することができます。

□ Azure Machine Learning
Azure Machine Learning は機械学習モデルをコーディングするための開発プラットフォーム

□ SLA
単一のAzure仮想マシンにおいては、SLAが <99.95%となった場合は10％のクレジットがユーザーに提供

・SLA <99.95%となった場合は、10％のクレジットがユーザーに提供される。  
・SLA <99%となった場合は、25％のクレジットがユーザーに提供される。  
・SLA <95%となった場合は、100％のクレジットがユーザーに提供される。   

□ Azure Stack
Azure StackはAzureをユーザー独自のオンプレミス環境にプライベートクラウドとして構成するためのサービス
オンプレミスにAzure展開が可能

□ Microsoft Developer Network（MSDN）
Microsoft Developer Network（MSDN） は、マイクロソフトの開発者ネットワークであり同社が開発者向けに提供するサービスの総称
従量課金制のサブスクリプションを設定することでMSDN Platformsにアクセスできる。

□ Azure Cost Management
Azure Cost Management のコストアラート機能はAzureのコストが閾値に達すると自動的にアラートを送信します。その際は、ユーザーが事前に条件を設定してアラートの閾値を定義することができます。

□ Microsoft Service Trust Portal
Azureサービスの監査レポートとコンプライアンス関連情報を確認できる。
マイラブラリ機能ではドキュメントとリソースを一箇所に保存できる。
ユーザは外部監査人によって生成された監査レポートをダウンロードすることができる

Microsoft Service Trust Portal では、Microsoft クラウド サービスのセキュリティ、
プライバシー、コンプライアンスクラウドに関する情報が提供されています。
ここを参照することで、Azureサービスの監査レポートやコンプライアンス関連情報を確認することができます。

□ トラストセンター
Azureのセキュリティ基準を把握することができる

□ Microsoft Defender for Cloud
Azureリソースがセキュリティ基準を満たしているか評価できる
コンプラインアンスダッシュボードを利用することで、セキュリテイ評価とコンプライアンスレポートをダウンロードできる

□ サポートリクエスt
二つのサブスクリプションを一つにすることはできない

□ Help and Support
Azureリソース全体についてメンテナンス観覧のイベント計画をリストで確認できる

□ Azure Activity Log
過去１４日間に特定の仮想マシンをオフにしたユーザーを表示できる

□ Azure Application Inshight
Webアプリの異常やパフォーマンスを検出した診断する

□ リージョン間の通信
全て無料。可用性セット間の通信は2023年7月より一部有料
異なるリージョン間はVNetピアリングで接続される。
AZ内は専用線で接続

□ Azure Information Protection
Azure内のドキュメントを保護する機能。
ユーザーが保存しているファイルなどラベルを不要することで、ドキュメントやメールの検出、分類及び保護できる

□ Azureストレージアカウントの暗号化
Azureは独自の暗号化サービスを利用してストレージアカウントを暗号化する

□ Azure Logic App
コード無しにワークフローを生成できる。
これを利用してメール通知を自動的に送信するアプリケーションを生成できる

□ Amazon Cost Management
過去三ヶ月の仮想マシン使用状況がわかる

□ ネットワークセキュリティグループ
仮想ネットワークのサブネットやネットワークインターフェイス関連づけてトラフィックを制御
→ 仮想ネットワークに関連づけるわけではない

□ 課金管理者
サブスクリプションの譲渡ができる

□ Azure 使用制限
設定したクレジット額を釣果した時点で利用を自動停止することで、想定外の支出が発生しないようにする予防機能

□ Containerサービスの移行先
Azure Container Instances はAzure 環境内に Docker コンテナーをオンデマンドで実行するサービス
Azure App Service で実行されるようにカスタムコンテナーを構成することができます。

□ DBの移行
通常はAzure SQL Databaseを利用するがが
共通言語ランタイム（CLR）はAzure SQL Databaseでは利用できないためAzure SQL Managed Instancewor利用する

□ Azure Policy Initiative
複数のポリシーをまとめて管理する機能。ポリシーのグループ管理ができる

□ Enterprise Agreement Subscription
250台以上のPCを保有する職場、学校などの組織が効率的にマイクロソフトのライセンスを購入できるプログラム
利用するためには AzureエンタープライズポータルにおいてMicrosoft Azureアカウントを追加する必要がある

□ Azure Advisor
バックアップされていない仮想マシンを識別して、バックアップの有効かを推奨してくれる
ユーザーは高信頼性、セキュリティ、オペレーショナル エクセレンス、パフォーマンスの向上、コスト改善を実現
全ての機能が無料で提供される

□ Azure プライベートDNS
Azure内の仮想ネットワークの信頼性が高く安全なDNSサービス
仮想ネットワーク内のドメインを名前解決するためのDNS機能、仮想ネットワークに関連づけて利用

□ Azure File Sync Agent
オンプレミス環境上のストレージにあるデータをAzureに同期できるサービス
Azureに保存しているデータをオンプレミスなどのWindows Severにキャッシュすることが可能
Windows Serverにエージェントをインストールする必要がある

□ Microsoft Online Services
「プライバシーに関する声明」を確認することで、Microsoftがユーザから収集したデータの利用目的などを確認できる

□ ページBlob
非マネージドディスク。ランダムアクセスに対応しており、Azure VMに利用できるストレージタイプ

□ Azure Synapse Analytics
Azure SQL Databaseと連携してデータハウスを構築できる
